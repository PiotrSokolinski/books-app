## Description

React application that provides a simple user interface for a book management application.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# watch development mode
$ npm run dev

# production mode
$ npm run prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run cypress:run

# test coverage
$ npm run test:coverage
```
