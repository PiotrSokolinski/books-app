## Description

NestJS application that provides a REST API for a simple book management application.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# smoke tests
$ npm run test:smoke

# test coverage
$ npm run test:cov
```
